# 🤖 AI Code Intelligence Platform

An AI-powered developer assistant that understands software repositories, analyzes source code, explains programming logic, generates documentation, and answers developer queries using Large Language Models.

The platform uses **Code Retrieval Augmented Generation (Code-RAG)** to provide intelligent insights from large codebases.

# 🚀 Project Overview

Modern software projects contain thousands of lines of code, making understanding and debugging difficult.

This project creates an AI assistant that can:

- Understand code repositories
- Search code using natural language
- Explain complex functions
- Generate documentation
- Identify possible bugs
- Answer programming questions
